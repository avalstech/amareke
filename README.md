# Amareke – Website (Vite + React + Tailwind)

Production ready marketing site plus a lightweight Live Demo page that simulates the Creator Studio experience.

## Quick start

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Pages

• Home: `/`
• Product: `/product`
• Pricing: `/pricing`
• Docs: `/docs`
• Live demo: `/demo`
• About: `/about`
• Contact: `/contact`

## Deployment

This is a standard Vite build.

• Build command: `npm ci && npm run build`
• Output directory: `dist`

AWS Amplify Hosting works well with these settings.
